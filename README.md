Overview
This report provides a sales performance dashboard built on the Sample - Superstore dataset. It delivers a high-level view of revenue, profitability, and sales distribution across product sub-categories and geographic regions.

How to Use

Open sales_data_analysis.pbix in Power BI Desktop (March 2026 or later recommended).
If prompted, refresh the data source to reconnect to the Superstore dataset.
Use the bar chart visuals to filter the page — clicking a region or sub-category will cross-filter all other visuals.
